# Prompt Examples

Use these examples as starting points. Keep private data out of prompts unless your app has the right user consent and data handling controls.

## 1. Review an API integration plan for reliability risks.

```text
Review an API integration plan for reliability risks.
```

## 2. Convert a support ticket into a concise engineering bug report.

```text
Convert a support ticket into a concise engineering bug report.
```

## 3. Summarize a code review thread into action items.

```text
Summarize a code review thread into action items.
```

## Practical Notes

- Start with a small, repeatable test before putting the model behind a user-facing workflow.
- Save the model name and request settings with each result.
- Keep generated output reviewable before it is published or sent to customers.
